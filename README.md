## 📖 About The Project

This is a **Full Stack Dashboard Application** built for **BotMakers Pvt Ltd** as part of the **Full Stack Developer Intern** assignment.

The application serves as a dashboard for **"The Arena"** - a gaming platform where users can:
- View active arena games and competitions
- Browse episodes and content
- Check national rankings of players
- Stay updated with the latest gaming events

The design is based on the Figma specifications provided by BotMakers, featuring a modern dark theme with gold accents.

---

## ✨ Features

### 🎯 Core Features
- ✅ **Responsive Dashboard** - Works on desktop, tablet, and mobile
- ✅ **Arena Games Display** - View active games with player counts and prizes
- ✅ **Episodes Section** - Browse available episodes with view counts
- ✅ **National Ranking Table** - View top players with rankings and scores
- ✅ **Dynamic Data Loading** - Data fetched from RESTful API

### 🎨 UI Features
- ✅ **Dark Theme** - Modern #1A1919 dark background
- ✅ **Gold Accents** - #FFD700 brand color
- ✅ **Smooth Animations** - Hover effects and transitions
- ✅ **Sticky Navigation** - Fixed header on scroll
- ✅ **Interactive Cards** - Hover effects on arena and episode cards

### 🔧 Technical Features
- ✅ **RESTful API** - Built with Python Flask
- ✅ **CORS Enabled** - Cross-origin resource sharing
- ✅ **Modular Code** - Clean separation of concerns
- ✅ **Error Handling** - Graceful error handling on API calls
